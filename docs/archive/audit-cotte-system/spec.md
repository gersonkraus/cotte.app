# Especificação: Auditoria Completa do Sistema COTTE

## Objetivo
Realizar uma auditoria técnica e de produto profunda no sistema COTTE para encontrar erros de lógica de negócio, fluxos desconectados entre frontend e backend, problemas de UX/layout, processos incompletos e inconsistências de mercado.

## Entregáveis Esperados
1. **Mapa dos Fluxos**: Um mapeamento detalhado da jornada do usuário e dos dados através do sistema.
2. **Lista de Erros Críticos**: Identificação de bugs ou falhas de lógica bloqueantes.
3. **Pontos Desconectados**: Falhas de integração entre ações do Frontend (UI) e respostas/processamento do Backend (API).
4. **Problemas de UX**: Levantamento de atritos na experiência do usuário, inconsistências de interface e layout.
5. **Plano Priorizado de Correção**: Um plano de ação estruturado por impacto e esforço técnico para resolver os problemas encontrados.

## Escopo de Análise
- **Backend**: FastAPI, modelagem do banco de dados SQLAlchemy, validações Pydantic.
- **Frontend**: Vanilla JS, estrutura HTML5/CSS3, reatividade e feedback visual.
- **Integrações Chave**: IA (Claude), envios via WhatsApp, PDFs e e-mails.
- **Fluxo de Negócio**: Desde a entrada de dados (captação), geração do orçamento, aprovação e gestão financeira.
