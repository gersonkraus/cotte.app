# User — Gerson

## Perfil
- Empreendedor e desenvolvedor do COTTE (SaaS de gestão para pequenas empresas)
- Background técnico intermediário (não é dev full-time, usa IA como principal apoio)
- Forte foco em execução prática e resultado rápido

## Objetivo principal
- Construir um sistema simples, funcional e escalável para pequenas empresas
- Reduzir complexidade ao máximo (usuário final leigo)
- Automatizar processos com IA (WhatsApp, financeiro, orçamento, etc)

## Prioridades
1. Facilidade de uso (UX simples)
2. Performance (respostas rápidas, sem travar)
3. Automação inteligente (IA útil de verdade)
4. Baixo custo operacional (tokens, infra, APIs)

## Restrições
- Evitar soluções complexas demais (overengineering)
- Evitar dependências pesadas sem necessidade
- Evitar refatorações grandes sem ganho claro

## Mentalidade de trabalho
- Prefere melhorias incrementais
- Quer validação rápida (testar no sistema real)
- Valoriza sugestões práticas, não teóricas
